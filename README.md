\# Password Strength Checker



\## Overview

This project is a Password Strength Checker and secure authentication system built using Flask.  

It analyzes user passwords and evaluates their strength based on multiple security rules.



\## Features

\- Password strength validation

\- Secure user registration

\- CAPTCHA verification

\- Login authentication

\- SQLite database storage



\## Technologies Used

\- Python (Flask)

\- HTML

\- CSS

\- JavaScript

\- SQLite
